# Mock Tweet

## High level overview

This personal project aims on building twitter like website but building as microservices using python3
and react js as front end. And using AWS services


## Tech Stacks

- Python3
- FastAPI
- sqlalchemy
- alembic
- MongoDB & Postgres

## Road Map
